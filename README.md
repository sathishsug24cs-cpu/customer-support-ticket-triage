# Customer Support Ticket Triage System

## Week-1 Internship Project

This project builds a rule-based customer support ticket triage system.
It reads customer messages, cleans text, classifies issues, assigns priority,
calculates SLA, and generates a manager-ready report.

## Tools Used
- Python
- Google Colab
- GitHub
- Pandas

## Features
- Text cleaning using regex
- Issue classification (PAYMENT, LOGIN, DELIVERY, REFUND, BUG, GENERAL)
- Priority assignment (P0â€“P3)
- SLA and due time calculation
- Handles large real-world dataset (2.8M records)

## Dataset
Customer Support on Twitter (Kaggle)

## Output
Final processed ticket data exported as CSV.
